20 HTML Attributes & Elements

SL No. Attribute Name Elements Descriptions
1 accept <form>, <input> List of types the server accepts, typically a file type.
2 action <form> The URI of a program that processes the information submitted via the form.
3 autofocus <button>, <input>, <keygen>, <select>,<textarea> The element should be automatically focused after the page loaded.
4 border <img>, <object>,<table> The border width.
5 color <basefont>, <font>,<hr> This attribute sets the text color using either a named color or a color specified in the hexadecimal #RRGGBB format.
6 cols <textarea> Defines the number of columns in a textarea.
7 colspan <td>, <th> The colspan attribute defines the number of columns a cell should span.
8 controls <audio>, <video> Indicates whether the browser should show playback controls to the user.
9 datetime <del>, <ins>, <time> Indicates the date and time associated with the element.
10 headers <th>, <td> IDs of the <th> elements which applies to this element.
11 integrity <link>, <script> Security Feature that allows browsers to verify what they fetch.
12 ismap <img> Indicates that the image is part of a server-side image map.
13 list <input> Identifies a list of pre-defined options to suggest to the user.
14 manifest <html> Specifies the URL of the document's cache manifest.
15 ping <a>, <area> Provides a hint to the user of what can be entered in the field.
16 loop <audio>, <bgsound>,<marquee>,<video> Indicates whether the media should start playing from the start when it's finished.
17 max <input>, <meter>, <progress> ndicates the maximum value allowed.
18 sizes <link>, <img>, <source> Defines Sizes of html elements
19 target <a>, <area><form>, <base> Defines target of html page after click.
20 width <canvas>, <embed>, <iframe>, <img>, <input>, <object>, <video> For the elements listed here, this establishes the element's width.
